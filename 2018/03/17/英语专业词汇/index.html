发表于 2018-03-17 | 分类于 杂项

HTTP HyperText Transfer Protocol，超文本传输协议
HTTPS Hyper Text Transfer Protocol over SecureSocket Layer，是以安全为目标的 HTTP 通道，HTTPS 在HTTP 的基础下加入SSL 层。
HSTS 全称 HTTP Strict Transport Security，HTTP严格传输安全，是一个Web安全策略机制。
DNS Domain Name System 域名解析系统
CDN Content Delivery Network，即内容分发网络
function 函数
functional 函数式
method 方法。js中指对象中定义的函数，赋值给对象属性的函数。
programme 编程
FP Functional Programming 函数式编程
OOP Object-oriented Programming 面向对象编程
refactor 代码重构。指对软件代码做任何更动以增加可读性或者简化结构而不影响输出结果。名词是refactorings。
FSC: functional stateless components 无状态组件，来自React，使用函数定义一个组件。
lifecycle: 生命周期
side-effects: 副作用
Browser 浏览器
API Application Programming Interface，应用程序编程接口
Abstraction 抽象
reusability 复用性
logic 逻辑
concept 概念 思想
feature 特性
backward compatible 向后兼容
bloated 臃肿
solutions 解决方案
boundary 边界
syntax 语法，句法
destructuring 解构
Object destructuring 对象解构
prevent 阻止 预防
ternary operator 三元操作符 ？：
spread operator 展开操作符 …
pitfall 陷阱
theme 主题
producer 生产者
consumer 消费者
crash 使奔溃 crash your application
performance issues 性能问题

IIFE 立即调用函数表达式(Immediately Invoked function Expression)
initially 起初
execute 执行
plain text 纯文本
In this aspect 从这个角度看
has no relation to 和……无关
embed 嵌入的
basics 核心 基本原理
QR code 二维码
unrealistic 不切实际的
contender 竞争者
giant 巨头 巨人
DCS Distributed Cache Service, 分布式缓存服务
WH Website Hosting, 静态网站托管, 是云开发提供的一项服务
Hosting 托管
appeal 呼吁

reimagine 再定义的

make sense 讲得通 言之有理
syntactic sugar 语法糖
GPU (=Graphic Processing Unit) 浏览器的图形处理器
GUI (Graphical User Interface) 图形用户接口. 浏览器采用图形方式显示的计算机操作用户界面。
AWS 亚马逊云计算服务
silver bullet 直译成“银弹”，私以为译成“万金油”（褒义）更接地气一些。
