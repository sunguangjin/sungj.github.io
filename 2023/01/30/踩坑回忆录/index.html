防盗链和反防盗链

  • 现象：图片地址浏览器能访问，通过 img 标签访问 404。
  • 防盗链：通过请求头中的 Referer 或者签名，网站可以检测目标网页访问的来源网页，如果是资源文件，则可以追踪到显示它的网页地址。 一旦检测到来源不是本站，即进行阻止或者返回指定的页面。
  • 反防盗链：通过设置 referer 为空进行绕过防盗链。
  <img
    referrerPolicy="no-referrer"
    src="https://lmg.jj20.com/up/allimg/4k/s/01/2109241303431U3-0-lp.jpg"
  />

字体加粗产生抖动

  • 问题：通过font-size字体加粗会改变整体宽度，交互体验上有抖动。
  • 解决：通过其他方式实现加粗效果，比如字体描边，阴影。
  /* 字体描边（推荐）*/
  div:hover {
    -webkit-text-stroke: 0.3px currentColor;
  }
  /* 字体阴影（不推荐，有点模糊）*/
  div:hover {
    text-shadow: 0 0 .5px currentcolor;
  }
